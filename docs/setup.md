# Setup

## Install / Update Software

```bash
# Update Homebrew
brew update

# Verify Homebrew health
brew doctor

# Install RBenv / Ruby-Build
brew install rbenv
brew install ruby-build

# ...Or Upgrade RBenv / Ruby-Build
brew upgrade rbenv
brew upgrade ruby-build
```

## Install Ruby

```bash
rbenv install
```

## Install Gems

```bash
bundle install
```
